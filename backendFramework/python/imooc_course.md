# Python Flask 全流程全栈项目实战

## 技术栈

* 前端：Vue.js
* 后端：Flask、jinja2、SQlAlchemy
* 网络: HTTP
* 系统: Linux
* 数据库：MySQL
* 缓存：Redis
* 队列：RabbitMQ
* 认证：JWT
* 监控：Sentry
* 测试：pytest
* 部署：Docker

## 学习目标

* 掌握程序设计思路
* 掌握Flask框架的使用
* 掌握jinja2模板的使用
* 能够独立完成项目的后端开发
* 能够独立完成独立完成市面上全部的博客以及社区类项目

## 快速学习的核心方法以及六个阶段

* 先敲代码模仿后再理解
* 重塑思维模式，树立编程思想

1. 阶段一：体系化知识的学习
2. 阶段二：项目模仿开发
3. 阶段三：项目 BUG 修复
4. 阶段四：项目未完成的功能开发
5. 阶段五：看上哪个项目，按照官网文档进行开发
6. 阶段六：发现身边的不便利，寻求其共性，自挖掘需求，进行系统的开发
